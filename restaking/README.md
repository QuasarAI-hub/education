## Restake Mathematics ğŸ“Š

### Compound Interest Formula
```
A = P(1 + r/n)^(nt)
Where:
A = Final amount
P = Principal (initial stake)
r = Annual interest rate
n = Number of times compounded per year
t = Time in years
```
